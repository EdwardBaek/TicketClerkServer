# Tecket Clerk

## About
The Server for Project [Ticket Clerk]()



## Stack
- Node.js
- Express 4
- Postgresql



## Install Server
```bash
clone git https://github.com/EdwardBaek/TicketClerkServer.git
npm i -y
```



### Modify Environment
Rename **.env.sample** fiel to **.env** and modify value
```
DATABASE_URL=yourDataBaseUrl
```
